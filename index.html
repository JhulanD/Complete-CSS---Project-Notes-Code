Chapter 1: Introduction to CSS

Learn the fundamentals of CSS and how to apply styles to HTML elements.

Selectors

Learn how to select HTML elements using CSS selectors.

  • Universal Selector: *
  • Class Selector: .class-name
  • ID Selector: #id-name
  • Element Selector: element-name
  • Descendant Selector: parent-element child-element
  • Adjacent Sibling Selector: element-name + element-name
  • General Sibling Selector: element-name ~ element-name
  • Attribute Selector: [attribute-name]
